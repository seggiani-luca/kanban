# kanban
Implementazione di server e client in C per un protocollo di gestione di [Kanban](https://en.wikipedia.org/wiki/Kanban).

## Compilare
Si rende disponibile un Makefile per la compilazione con `make`:
```
# compila il server
make server

# esegui il server
make run_server

# ripulisci gli oggetti
make clean
```
